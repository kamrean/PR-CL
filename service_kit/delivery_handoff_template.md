# 交付移交模板

## 使用目标

这份模板用于在交付结束时，把结果正式交给客户，并顺手为下一步合作留入口。

## 模板

```text
这次交付已经整理好了，核心内容如下：

1. 一句话结论：
[填写]

2. 最重要的 3 个问题：
[填写]

3. 最建议先做的 3 个动作：
[填写]

4. 风险等级：
[填写]

5. 是否有需要你确认的事项：
[填写]

详细报告我已经按交付包整理好，你可以优先先看 summary 和 next steps。

如果你希望，我下一步可以继续帮你做：

- 仓库体检后的 GitHub Actions 落地
- 当前红灯的后续修复建议或修复 PR
- PR 初审托管
- 月度 PR/CI 托管
```

## 发送前提醒

1. 确保附件或内容已脱敏
2. 确保 summary 在最前面
3. 确保客户能一眼看到下一步
4. 不要一次发太多无解释的文件
