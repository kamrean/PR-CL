# 案例展示素材检查表

## 使用目标

在把某个案例公开展示前，用这份清单确认素材是否齐全、是否安全。

## 素材清单

### 必备

- [ ] 案例名称
- [ ] 客户场景一句话
- [ ] 我们做了什么
- [ ] 结果是什么
- [ ] 为什么有价值

### 可选

- [ ] Before / After
- [ ] 脱敏日志片段
- [ ] GitHub Actions 截图
- [ ] PR 页面截图
- [ ] 交付包 summary 截图

## 安全检查

- [ ] 已确认客户允许展示
- [ ] 已确认案例可匿名展示
- [ ] 没有 token
- [ ] 没有 key
- [ ] 没有 cookie
- [ ] 没有私钥
- [ ] 没有真实用户个人信息
- [ ] 没有生产配置

## 展示质量检查

- [ ] 客户看 30 秒能看懂
- [ ] 不是只有过程没有结果
- [ ] 不是只有结论没有证据
- [ ] 不是堆一大段日志
- [ ] 能让下一位客户想起自己的问题
