# GitHub Actions 升级矩阵

## 基础版

适合：

- 没有 CI 的仓库
- 第一次做工程化的团队

包含：

- install
- test
- lint
- build
- 最小权限

## 稳定版

适合：

- 已经有基础 CI
- 希望减少重复失败和定位时间

包含：

- 基础版全部内容
- 更清晰的失败分层
- 更严格的 artifact 白名单
- 更好的命令和环境说明

## 团队版

适合：

- PR 数量稳定
- 需要更强协作约束

包含：

- 稳定版全部内容
- branch protection 建议
- required checks 建议
- PR 审核流程约束

## 扩展版

适合：

- 想继续往自动化和托管推进

包含：

- 更丰富的缓存策略
- 矩阵测试
- 前端冒烟测试
- 安全扫描

## 使用建议

第一单不要直接卖扩展版。

推荐顺序：

1. 基础版
2. 稳定版
3. 团队版
4. 扩展版
